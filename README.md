Superstore Sales Performance Dashboard

Project Overview
This project presents an interactive sales dashboard built using Power BI to analyze sales performance by region, category, and month. The objective was to visualize trends and extract business insights from the Superstore dataset.

Tools Used
- Power BI
- Microsoft Excel (for basic data formatting)
- Dataset: Sample Superstore

Dataset Details
The dataset contains transactional sales data including:
- Order Date
- Region
- Category
- Sales
- Profit
- Segment
A Month-Year column was created to analyze sales trends over time.
